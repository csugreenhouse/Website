Project Overview

Summary

This project focuses on the design of an automated greenhouse monitoring system that uses image-based height estimation to eventually continously adjust water and light levels to optimze plant growth. This system uses embedded camera systems, computer vision, potentially machine learning, and backened software to collect and process images. With the data we collect from the plants, such as plant height, plant general health, we then hope to proactivley adjust environmental parameters, such as dynamical changing the lighting schedule.

Project Background

The project currently has fan, light, and water control, as well as a ph sensor and electrical conductivity sensor. It currently successfully runs water throughout the greenhouse. Plants have been successfully grown inside of the greenhouse.

Constraints

The greenhouse is a self contained unit, that is roughly the size of a dresser. It is powered by a standard wall outlet. We want the machine learning model to be locally o the rasberry pi in the greenhouse.
